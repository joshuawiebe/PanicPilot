# PanicPilot

Mario Kart ähnliches Coop Spiel, wo einer lenken kann bzw. das Auto steuert und der andere die Karte sieht. Sie müssen Kommunizieren um ans Ziel zu gelangen. Später könnte man auch noch Items, Buffs oder anderen Möglichkeiten, wie z.B. das der eine Lenkt und der andere Gas gibt implementieren.
